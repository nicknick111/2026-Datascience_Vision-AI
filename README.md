# 🚀 드론 부품 표면 결함 검출 Vision AI 시스템 — 마스터 가이드

> **단일 모델의 한계를 극복하고 고도화된 Two-Stage OBB 파이프라인으로 완성된 실시간 품질 검사 시스템 통합 안내서**

본 문서는 프로젝트의 전체 아키텍처 흐름, 엔지니어링 시행착오, 그리고 최종 완성된 시스템의 구조를 한눈에 파악할 수 있도록 작성된 **최상위 통합 가이드(Master Guide)**입니다. 하위 폴더인 `failed_attempts/`와 `two_stage_pipeline/`의 관계를 설명하고, 핵심 기술 문서들을 안내합니다.

---

## 1. 프로젝트 핵심 요약

본 프로젝트는 컨베이어 벨트 위를 지나가는 드론 부품(C4 커넥터, 통신 모듈, 전원 공급 장치)의 표면 결함을 실시간으로 탐지하고 정밀하게 분류하는 Vision AI 시스템입니다. 

* **목표 클래스:** `Defect`(결함) / `Normal_C4` / `Normal_Comm` / `Normal_Supply`
* **핵심 파이프라인:** **YOLOv8n-OBB** (공간 탐지 및 회전 보정) × **EfficientNet-B0** (정밀 미세 질감 분류)
* **하드웨어 환경:** RPC-20F 카메라 (최신 렌즈 왜곡 보정 적용)
* **소프트웨어 스택:** Python 3.11, PyTorch, FastAPI, OpenCV, Ultralytics

---

## 2. 아키텍처 진화 과정 비교 (V1 vs V2)

프로젝트는 단순히 코드를 작성하는 것을 넘어, 문제 발견과 원인 분석을 거쳐 구조를 개선하는 반복 주기(Iteration)를 거쳤습니다. 아래는 초기 설계와 최종 시스템의 핵심 차이점입니다.

| 비교 항목 | Phase 1. 단일 모델 접근 (`failed_attempts/`) | Phase 2. Two-Stage 파이프라인 (`two_stage_pipeline/`) |
| :--- | :--- | :--- |
| **객체 탐지 방식** | 일반 BBox (배경 노이즈 대량 혼입) | **OBB (Oriented Bounding Box)**를 통한 정밀 회전 각도 추출 |
| **분류 엔진** | YOLOv8 분류 헤드 (미세 특징 추출 한계) | **EfficientNet-B0** 전이학습 (결함 패턴 집중 학습) |
| **데이터 증강** | 3D 시뮬레이션 환경 (도메인 갭 발생) | 실제 촬영 이미지 기반 **7대 OpenCV 고급 증강 기법** |
| **서버 통신 구조** | 동기식 While 루프 (FastAPI 타임아웃 발생) | **Non-blocking 스레드 분리** 및 Polling/비동기 큐 전송 |
| **이미지 전처리** | 단순 비율 강제 리사이즈 (형상 왜곡) | 고급 필터링(CLAHE, Unsharp) + **Letterbox 패딩** (비율 보존) |

## 3. 리포지토리 폴더 구조 및 내비게이션

본 프로젝트는 "실패를 통한 성장"과 "최종 완성된 아키텍처"를 분리하여 보존하고 있습니다.

### 📁 `failed_attempts/` (시행착오 및 R&D 아카이브)
단일 YOLO 모델로 모든 것을 해결하려 했던 시도와, 한계에 부딪히며 진행한 다양한 실험 코드들이 보존되어 있습니다.
* **주요 내용:** 카메라 연동 기초, 3D 데이터 증강 시도, 모놀리식(Monolithic) 서버 통합 시도, 디버깅 유틸리티.
* **의의:** 이 폴더의 코드들은 실패작이 아닌 최종 아키텍처를 설계하기 위한 **지식 은행**입니다. 여기서 도출된 렌즈 왜곡 보정값과 비동기 스레드 개념이 최종 파이프라인에 그대로 계승되었습니다.

### 📁 `two_stage_pipeline/` (최종 프로덕션 시스템)
YOLOv8n-OBB와 EfficientNet-B0의 역할을 완벽히 분리하고, FastAPI를 통해 안정적인 통신망을 구축한 최종 코드 베이스입니다.
* **주요 내용:** OBB 기반 시드 학습 및 파인튜닝, 클래스 불균형 해소 알고리즘, 2단 정확도 필터 시스템, 메인 추론 엔진(`vision_ai_main.py`), 비동기 통신 서버(`api_server.py`).
* **의의:** 실시간 프레임 드랍을 막기 위한 백그라운드 Active Learning 큐 시스템 등 산업 현장의 요구사항을 반영한 고도화된 엔지니어링 결과물입니다.

## 5. 비전 AI 리드 파트의 핵심 엔지니어링 성과

1.  **3D 투시 왜곡 한계 극복:** 컨베이어 위 임의의 각도로 놓이는 부품을 인식하기 위해 OBB를 도입하여, 배경 노이즈를 제거하고 효율적인 224x224 규격화를 달성했습니다.
2.  **스레드 안전성(Thread-Safety) 확보:** 메인 AI 추론 루프와 API 스트리밍(MJPEG) 루프 간의 자원 충돌을 `threading.Lock()` 메커니즘으로 완벽히 제어했습니다.
3.  **Active Learning 자동화:** 판정 신뢰도 `50% ~ 80%` 구간의 불확실 데이터를 백그라운드 큐(Queue)로 비동기 저장하여, 메인 시스템의 I/O 딜레이 없이 차세대 모델 학습 데이터를 자동으로 적재하는 루프를 구축했습니다.
4.  **카메라 하드웨어 캘리브레이션 최신화:** 시스템 구동 환경에 맞춰 ArduCAM B0454C의 내부 파라미터(Intrinsic Parameter)를 측정하고 런타임 보정 맵으로 통합했습니다.
